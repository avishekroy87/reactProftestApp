# reactProftestApp

AI-powered React application for conducting professional technical interviews using configurable prompt templates and LLMs. Supports interactive Q&A, adaptive follow-up questions, candidate evaluation, and reusable prompt engineering for React, Python, FastAPI, AWS, GenAI, and other technical domains.

## Scripts

- `npm run dev` starts the development server.
- `npm run build` creates a production build.
- `npm run preview` previews the production build.
- `npm run lint` runs Oxlint.
